# Project 2 — Exploratory Data Analysis (EDA)
**DecodeLabs Data Analytics Internship, Batch 2026**

## What this is
Digs into the cleaned e-commerce dataset from Project 1 to find patterns,
trends, and outliers — descriptive stats, distribution shape, outlier
detection (IQR method), revenue breakdowns, and a correlation analysis.

## Files
| File | What it is |
|---|---|
| `eda_analysis.py` | Full analysis script (pandas, seaborn, matplotlib) |
| `EXECUTIVE_SUMMARY.md` | Findings written up for a non-technical stakeholder |
| `cleaned_dataset.xlsx` | Input data (cleaned in Project 1) |
| `outputs/` | Charts (PNG) + stats tables (CSV) generated by the script |

## How to run it
```bash
pip install pandas matplotlib seaborn openpyxl
python3 eda_analysis.py
```

## Key finding
41.4% of all orders are Cancelled or Returned — the single most important
number in this dataset. Full breakdown in `EXECUTIVE_SUMMARY.md`.
